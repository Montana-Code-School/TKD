<h2>Instructions for startup</h2>
<h4>Run all commands from root folder</h4>
<ul>
  ```
  npm install
  ```
  <li>In ```PaymentBackend/constats/stripe.js``` change ```sk_test_MY_SECRET_KEY``` to your Stripe secret key.
  <li>```node PaymentBackend/index.js```</li>
  <li>```npm run start```</li>
  <li>Test it out! :)</li>
</ul>
