<h2>Instructions for startup</h2>
<h4>Run all commands from root folder</h4>

  ```npm install```

In ```PaymentBackend/constats/stripe.js``` change ```sk_test_MY_SECRET_KEY``` to your Stripe secret key. <br>
  ```node PaymentBackend/index.js``` <br>
  ```npm run start``` <br>
  <p>Test it out! :)</p>
